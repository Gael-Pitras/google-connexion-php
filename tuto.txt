Étape 1 : Ouvrez Google API Console.
https://console.cloud.google.com/

Étape 2 : Sélectionnez l'option Sélectionner un projet.

Étape 3 : Cliquer sur l'option "Nouveau projet", pour créer un projet.

Étape 4 : Chercher dans la bibliothèque "Google People API".

Étape 5 : Dans la gestion de l'api aller dans "Identifiants".

Étape 6 : Completer le form pour recuperer un "ID Client".
83781273492-ov1em5ma4lpom9o2ek70atcb9eonuh29.apps.googleusercontent.com

Etape 7: Installer composer sur votre machine :
https://getcomposer.org/download/

Etape 8: Installer la bibliothèque client Google PHP SDK:
composer require google/apiclient "^2.0"
